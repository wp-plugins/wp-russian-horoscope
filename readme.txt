== Description ==
Данный плагин выводит актуальный гороскоп на текущее число календаря. Имеет несколько категорий и генератор в админпанели.

Категории
1. Общий - общая характеристика дня основанная на лунном календаре и прогнозы для 12 знаков зодиака
2. Любовный - любовный гороскоп для 12 знаков зодиака
3. Мобильный - юмористический гороскоп для владельцев мобильных телефонов для 12 знаков зодиака
4. Автомобильный - юмористический гороскоп для владельцев автомобилей 12 знаков зодиака

Структура шорткодов
Код ruhoroscope, параметры TYPE и MARK

* TYPE - Тип гороскопа. Например: Любовный 
* MARK - Знак гороскопа. Например: Бизнецы

== Installation ==

1. Скачать и распаковать плагин в каталог плагинов WordPress (обычно `/wp-content/plugins/`).
2. Активировать плагин через меню 'Плагины' в вашем WordPress Admin.
